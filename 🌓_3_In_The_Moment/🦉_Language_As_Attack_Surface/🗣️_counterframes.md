## ⚙️ Deployment Pattern  

1. **Short + True + Repeatable** — ≤ 7 words; preserve subject and verb.  
2. **Mirror first, then correct.** E.g. “What they call *population adjustment* is forced displacement.”  
3. **Archive the moment.** Attach screenshot + link + watchlist ID.  
4. **Amplify through coalition.** Share exact counter-phrasing across aligned accounts.  
5. **Update monthly.** Add new pairs to both this file and the CSV glossary.

# 🧬 STAT Survivor Tools  
**First created:** 2026-05-25 | **Last updated:** 2026-05-25  
*Immediate stabilisation tools for moments of overload, disruption, exhaustion, or acute stress.*

---

## 🧭 Purpose  

STAT Survivor Tools are the lowest-bandwidth practical layer of the protocol.

This folder exists for moments where:
- cognition is reduced,
- stress is elevated,
- executive function is impaired,
- or immediate practical support is needed quickly.

The emphasis is:
- short actionable steps,
- minimal theory,
- low cognitive overhead,
- and immediate survivability.

---

## 📦 Typical Inclusions  

Files here may include:

- grounding exercises,
- emergency checklists,
- “do this first” guides,
- rapid evidence-preservation instructions,
- communication pause scripts,
- emotional regulation tools,
- immediate safety workflows,
- low-energy navigation aids,
- and cognitive offloading tools.

---

## 🚫 Exclusions  

This folder is **not** for:
- deep systems analysis,
- long-form theory,
- extensive political framing,
- high-abstraction governance discussion,
- or long-term strategic planning.

If a node requires sustained concentration or extended contextual understanding, it likely belongs in:
- 🌔 Small Steps, or
- 🌕 Long Strategies.

---

## 🧠 Routing Principle  

Ask:

> “Could somebody realistically use this while distressed, exhausted, overloaded, frightened, or cognitively impaired?”

If yes:
- it may belong here.

If not:
- route upward into higher-capacity layers.

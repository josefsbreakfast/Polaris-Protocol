## 🧩 Methods  

1. **Use only publicly available data** (grant databases, press releases, institutional repositories).  
2. **Aggregate by institution**, not by individual researcher.  
3. **Record provenance** for every data point (URL, access date).  
4. **Archive responsibly** — no scraped personal data or unpublished material.  
5. **Analyse comparatively** — look for systemic patterns, not isolated cases.

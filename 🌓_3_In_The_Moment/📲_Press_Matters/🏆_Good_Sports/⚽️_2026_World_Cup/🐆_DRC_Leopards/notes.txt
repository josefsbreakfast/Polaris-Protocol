# Drift Control Rule

For this folder, I’d use this rule:

```text
Every node must answer:
What is the public-facing story?
What is the system behind it?
What is known?
What is uncertain?
What should not be claimed?
Who is being made more human by this note?
```

And the anti-wobble rule:

```text
If a note starts making Congo into only suffering, route back to football, culture, language, humour, style, and self-representation.

If a note starts making Congo into only joy, route back to extraction, conflict, colonial history, and supply chains.

If a note starts making Western readers the centre, route back to Congolese people as subjects, not scenery.
```

That should keep the pack from drifting into either crisis-only or aesthetic-only.

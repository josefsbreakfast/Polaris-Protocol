### 1. Survivorship Is Created by System Design  
Survivors become the only continuous thread across:
- NHS  
- justice  
- safeguarding  
- universities  
- local authority  
- mental health  
- police

This is **structural**, not personal.

---

### 2. Survivors Accumulate System Knowledge  
Through repeated harms and attempts at repair, survivors:
- learn the architecture of systems  
- understand pathways better than staff  
- detect drift, contradictions, and omissions  
- see patterns invisible from inside silos  

Survivors become **unofficial system cartographers**.

### 3. Survivorship as a Long-Term State  
Because systems respond slowly:
- data harm persists for years  
- correction involves many agencies  
- silence-harm persists  
- reflex mismatches compound  

Survivorship becomes an *enduring administrative identity*.

### 4. Survivorship Is Not Chosen

- chronic case complexity  
- fragmented accountability  
- recursive harm  
- emotional burden  
- siloed confusion  

The survivor holds all the strands because nobody else can.

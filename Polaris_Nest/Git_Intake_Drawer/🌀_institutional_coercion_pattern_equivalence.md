## 3. How Bureaucratic Systems Produce These Patterns

Certain institutional structures generate coercion-equivalent dynamics *by design* or *by accident*:

### **A. Risk Inflation Loops**  
Multiple overlapping safeguarding frameworks all assume risk > 0.  
The user can never fully exit suspicion.

### **B. Asymmetric Power Distribution**  
Institutions can escalate, but users cannot de-escalate.

### **C. Metadata Echoes**  
Old flags, labels, or misclassifications persist beyond accountability structures.

### **D. Disjointed Actor Chains**  
The person interacting with you is not the person deciding about you.

### **E. Administrative Silence**  
Delays and non-responses function like relational stonewalling.

### **F. Behaviour-driven Feedback**  
Your attempt to correct the system is interpreted as further risk.

None of these require intent.  
All of them generate **coercive pattern equivalence**.

## How to Log
- **Timestamp (UTC):** exact time interference occurred.  
- **Platform / System:** where the event happened.  
- **Action Attempted:** what you were trying to do.  
- **Observed Outcome:** what actually happened (include screenshots if possible).  
- **Context:** any unusual factors (keywords, topic, tags, prior flags).  
- **Version Info:** app version, browser, device, or commit hash.  

Use consistent markdown or CSV so entries can be aggregated and analysed.
